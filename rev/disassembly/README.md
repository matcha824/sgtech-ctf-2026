# Name
Disassembly

## Description

Your final target is a fortress with multiple layers of security, where neither casual observation nor external surveillance will reveal anything useful. The only way in is to understand the building's architecture - the layout of its corridors, the mechanisms of its locks, and the patterns of its security systems. Once you comprehend how it was constructed, you can reverse engineer the escape route and use the security that the architects themselves built against them.

Can you dismantle the defenses and escape?

We recommend using a disassembler, like Ghidra or IDA, for this task, agent.
