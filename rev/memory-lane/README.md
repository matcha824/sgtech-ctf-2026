# Name
Memory Lane

## Description

Your second target is a secure facility where you can't simply walk in and take what you need, but you may not need to. Intelligence suggests that, while the building itself is impenetrable, the target sends the sensitive information we need out during specific operations. If you can observe that transmission from a distance at the right moment, you might catch the data in transit without ever setting foot inside.

Can you uncover the secret by watching from the shadows?

(Hint 1: You should not need to reverse engineer the encryption method)

(Hint 2: A debugger could work, but there are other ways to trace library calls as well)
