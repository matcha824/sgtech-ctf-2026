# Challenge
Man's Best Friend

# Solutions
1. Challenge presents a social media profile in an unauthenticated state. User is prompted to sign in. All information is available in the profile and posts.
	a. Click Log In
	b. Click Forgot Password
	c. Enter email found in About section
	d. Enter security question based on post history. There is a post about the profile user's childhood dog with the name tag visible
	e. Flag is shown as new password
2. Inspect web page and analyze source code to find encoded security key or flag/password.
3. Brute force password guessing. There's no time out on wrong attempts.
