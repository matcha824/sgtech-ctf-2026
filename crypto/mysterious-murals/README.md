# Mysterious Murals

## Description

Steve has a sudden interest in cybersecurity lately after hearing about a lot of vulnerabillity attacks on the internet. Turns out he's been experimenting with some encryption lately. While he's away you notice this secret message on this computer text file:

```
66cd7cf50f028cbea4c4cb46676ec10dc8046c0b43a941f7881f3a201123c5d8
```

You also notice on his filesystem in a folder called `/secret` there's an image called `secret-image.png`

Using this information, can you determine what this secret message says? 

**Flag format:** `sgctf{DECRYPTED_MESSAGE_CONTENT}`
You can paste the secret message as is into this format
